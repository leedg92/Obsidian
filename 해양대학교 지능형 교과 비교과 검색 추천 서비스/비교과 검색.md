

### 핵심역량(main_category) 종류 쿼리
- 
```
select * from im_eco_category_table  
where parent = '0';
```


### 핵심역량(sub_category) 종류 쿼리
```
select * from im_eco_category_table  
where parent = '1'[ main_category의 idx];
```


### 태그(tag) 쿼리
```
select * from im_eco_tag_table  
where latest_program != '0';
```
