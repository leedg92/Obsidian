## 📋 PNIT 현장 작업 절차

### 1. 파일 업로드

```bash
# USB나 네트워크로 3개 파일을 /root/ 에 복사
```

### 2. 체크섬 확인

```bash
cd /root
sha256sum -c apache-2.4.63-pnit-ready.tar.gz.sha256
```

### 3. 설치 실행

```bash
cd /root
./install-pnit.sh
```

### 4. Apache 시작

```bash
/usr/local/apache2/bin/apachectl start
ps aux | grep httpd
ss -nltp | grep :80
```

### 5. Tomcat 연동 확인

```bash
curl -I http://localhost
# Tomcat 응답 확인
```
