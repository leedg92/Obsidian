# 전송 직전 로그 포맷 대응표 (레거시 블록체인 vs 현재 카카오)

## 검색 키워드
- 레거시: `[BPA] [VBS] process.*SEND:`
- 현재: `_KAKAO_SEND`

## 대응표

| 이벤트                  | 레거시 (블록체인 전송전)                                                | 현재 (카카오 전송전)                        |
| -------------------- | ------------------------------------------------------------- | ----------------------------------- |
| GateIn               | `[BPA] [VBS] processGateIn SEND: {}`                          | `GATE_IN_KAKAO_SEND`                |
| GateOut              | `[BPA] [VBS] processGateOut SEND: {}`                         | `GATE_OUT_KAKAO_SEND`               |
| GateError            | `[BPA] [VBS] processGateIn/Out SEND: {}` (에러플래그 구분)           | `GATE_ERROR_KAKAO_SEND`             |
| CancelOut            | `[BPA] [VBS] processCancelGate SEND: {}`                      | `CancelOut_KAKAO_SEND`              |
| CopinoVerification   | `[BPA] [VBS] processCopinoVerification SEND: {}`              | `COPINO_VERIFICATION_KAKAO_SEND`    |
| CopinoDelete         | `[BPA] [VBS] processCopinoRemove SEND: {}`                    | `COPINO_DELETE_KAKAO_SEND`          |
| JobDone              | `[BPA] [VBS] processJobDone SEND: {}`                         | `JOB_DONE_KAKAO_SEND`               |
| EmptyConSwap         | `[BPA] [VBS] processEmptyConSwapResult SEND: {}`              | `emptyConSwap_KAKAO_SEND`           |
| EmptyConSwap M2Block | `[BPA] [VBS] AutoPickup processEmptyConSwapResult SEND: {}`   | `EMPTY_CON_SWAP_M2BLOCK_KAKAO_SEND` |
| NonLoc               | `[BPA] [VBS] processNonConLocation SEND: {}`                  | `NonLoc_KAKAO_SEND`                 |
| ChangeConLoc         | `[BPA] [VBS] processNonConLocation SEND: {}` (NonLoc과 동일 서비스) | `ChangeConLoc_KAKAO_SEND`           |
