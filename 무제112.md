# AAS API 호출 분석

> 대상: `https://tssapi.chainportal.co.kr/tss/aas/v1`
> 어댑터: `AASAdapter.java` → `AASPort` 인터페이스 구현
> 역할: TSS 오픈API용 중앙 인증 서버 (토큰 검증 + 콜백 URL 관리)

## 엔드포인트 요약

| #   | URL                           | Method | 용도            |
| --- | ----------------------------- | ------ | ------------- |
| 1   | `/tss/aas/v1/tokens/validate` | POST   | 액세스 토큰 유효성 검증 |
| 2   | `/tss/aas/v1/tokens/callback` | POST   | TMS 콜백 URL 조회 |

---

## 1. `/tokens/validate` — 토큰 검증

### 요청 (Request)

| 필드 | 타입 | 제약 | 설명 |
|------|------|------|------|
| `accessToken` | String | NotBlank, max 36 | 28자리 랜덤값 + 8자리 서명값 |

### 응답 (Response)

| 필드 | 타입 | 설명 |
|------|------|------|
| `result` | String | 성공 시 `"success"` |
| `resultCode` | String | 결과 코드 |
| `resultMessage` | String | 결과 메시지 |
| `data.entCd` | String (max 10) | 업체 코드 |
| `data.bizNo` | String (max 20) | 사업자등록번호 |
| `data.regId` | String (max 36) | 토큰 등록 ID |

### 호출처

| 호출 클래스 | 용도 |
|------------|------|
| `TssOpenApiAuthApplicationService` (174행) | Spring Security 인증 체인 — TSS API 요청마다 토큰 검증 |
| `AdminApplicationService` (49행) | 관리자 기능에서 토큰으로 업체 정보(entCd) 조회 |

---

## 2. `/tokens/callback` — 콜백 URL 조회

### 요청 (Request)

| 필드 | 타입 | 제약 | 설명 |
|------|------|------|------|
| `regId` | String | NotBlank, max 36 | 토큰 등록 ID |
| `tmsCallbackType` | TssCallbackTypeVO | NotNull | 콜백 유형 (아래 표 참조) |

### tmsCallbackType 종류

| 코드                     | 코드명                          | 설명               |
| ---------------------- | ---------------------------- | ---------------- |
| `GRP_STAT_CHNG_ALM`    | GRP_ORDER_STATUS_CHANGE      | 그룹오더 상태 변경 알림    |
| `GRP_VRF_RES_ALM`      | GRP_ORDER_VERIFY_RESULT      | 그룹오더 검증 결과 알림    |
| `GRP_VRF_RES_ALM_LIST` | GRP_ORDER_VERIFY_RESULT_LIST | 그룹오더 검증 결과 목록 알림 |
| `GRP_TML_EVENT_ALM`    | GRP_ORDER_TERMINAL_EVENT     | 그룹오더 터미널 이벤트 알림  |

### 응답 (Response)

| 필드 | 타입 | 설명 |
|------|------|------|
| `result` | String | 성공 시 `"success"` |
| `resultCode` | String | 결과 코드 |
| `resultMessage` | String | 결과 메시지 |
| `data.tmsCallbackUrl` | List\<String\> | 콜백 URL 목록 |

### 호출처

| 호출 클래스 | 용도 |
|------------|------|
| `TssGroupOrderVerificationService` (339행) | 그룹오더 검증 결과를 TMS에 콜백 전송 시 URL 조회 |
| `TssGroupOrderVerificationResultService` (168행) | 검증 결과 목록을 TMS에 콜백 전송 시 URL 조회 |
| `NotificationApplicationServiceImpl` (140행, 405행) | 터미널 이벤트/상태 변경 알림을 TMS에 콜백 전송 시 URL 조회 |
