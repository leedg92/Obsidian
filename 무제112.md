


컨테이너 타입 원본이 추가되었습니다.
추가로 ISO 6346 규격에 따라 타입 원본을 해석하여 사이즈와 카테고리를 분류하여 포함하였습니다.

## conType 패턴

| 그룹          | 형태       | 예시                          | 비중    | 파싱           |
| ----------- | -------- | --------------------------- | ----- | ------------ |
| ① 4자리 숫자    | `NNNN`   | `4510`,`2210`,`4500`        | ~95%  | 사이즈+타입       |
| ② ISO표준 영숫자 | `NN`+알파벳 | `45R1`,`22G1`,`42G0`,`L510` | ~0.3% | 사이즈+타입       |
| ③ 2글자 알파벳   | `AA`     | `GE`,`RF`,`FR`,`OT`,`TK`    | ~4.6% | 타입만 (사이즈 불가) |
| ④ 노이즈       | -        | `null`(문자열) 등               | 극소수   | 불가           |

# 사이즈

- ISO 6346: 1번째 글자 = 길이. (2번째 = 높이,  높이는 본 분류에서 사용하지 않으므로 "길이만" 사용, HC 구분 안 함)

| 1번째 글자                              | con_size           |
| ----------------------------------- | ------------------ |
| `2`                                 | `20` (20ft)        |
| `4`                                 | `40` (40ft, HC 포함) |
| `L`                                 | `45`               |
| `M`                                 | `48`               |
| `N`                                 | `49`               |
| 그 외(`9`,`5`,`G`,`R`,`F`,`O`,`T`...) | `''` (공란)          |


# 타입

타입은 원본 형태에 따라 
- 숫자형은 뒤2자리의 십의자리
- ISO 영숫자형은 3번째 글자
- 2글자 약어는 약어 자체
로 판별합니다

| 숫자형 (뒤2자리의 십의자리) | 알파벳형 (3번째 글자) | 2글자 약어 | → con_category   |
| ---------------- | ------------- | ------ | ---------------- |
| `0x`,`1x`        | `G`           | `GE`   | **DRY** (드라이/일반) |
| `3x`             | `R`           | `RF`   | **RF** (리퍼)      |
| `5x`             | `U`           | `OT`   | **OT** (오픈탑)     |
| `6x`             | `P`           | `FR`   | **FR** (플랫랙)     |
| `7x`             | `T`, `K`      | `TK`   | **TANK** (탱크)    |
| 그 외              | 그 외           | -      | **UNKNOWN**      |

참고사항으론
- 원본의 conType이 NULL인 경우 UNKNOWN으로 채워뒀습니다.
- 사이즈코드가 없는 원본(예:GE) 의 경우 con_size는 공란입니다.
