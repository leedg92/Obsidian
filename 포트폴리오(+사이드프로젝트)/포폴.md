# 기술 스택 & 프로젝트 이력

## 🛠 기술 스택

### Backend
- **Java** - Spring, Spring Boot 기반 서버 개발
- **Oracle DB, MariaDB, MySQL** - 데이터베이스 설계 및 운영
- **Apache Airflow** - 데이터 파이프라인 워크플로우 관리

### Frontend
- **JSP** - 전자정부 프레임워크 기반 웹 개발

### DevOps & Infrastructure
- **Linux, Ubuntu** - 서버 운영
- **Apache Tomcat** - WAS 구축 및 운영
- **Docker** - 컨테이너 기반 서비스 운영
- **GitLab** - 형상관리

## 💡 개발 원칙

- **철저한 문서화** - 개발 착수 전 ERD, API 명세서, 화면 정의서 작성을 통한 명확한 방향성 수립
- **단계적 검증** - 작은 단위 기능부터 체계적인 테스트와 교차 검증을 통한 시스템 안정성 확보
- **협업 중심** - "퇴근은 다같이 하자"는 철학으로 프로젝트 단계별 조율과 동료 업무 지원
