#### 로그 보는법
##### 운송오더(copino) 미수신
- accio 로그모니터링에 차량번호로 검색
	- 해당 데이터가 있으면 allcone 확인
	- 해당 데이터가 없으면 터미널에이전트 확인
		- 로그 봤는데 데이터 싸줬으면 블록체인 확인

##### 인수도증이 안나온다
- accio 로그모니터링에 차량번호 검색
	- 게이트진입 로그가 있으면 allcone 확인
	- 해당 데이터가 없으면 터미널에이전트 확인
		- 로그 봤는데 데이터 싸줬으면 블록체인 확인
