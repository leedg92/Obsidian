> 작성일: 2026-03-27

## 배경

- **현재**: 회원 데이터가 `chainportaldb.tb_a_user_m` 등 구 테이블에 저장
- **목표**: `chainportaldb.tb_member` 신 테이블을 직접 바라보도록 코드 변경
- **DB 위치**: 같은 chainportaldb (datasource 변경 불필요)

## 확인 완료 사항

1. **MBR_ID 채번** -> 랜덤6자(A-Z,0-9) + YYMM (예: "K3F9X22603"), 중복체크 필요
2. **AUTHRT_ID** -> ROLE_DRIVER = **28**
3. **JOIN_CHNL_CD_ID** -> 올컨e = **604**
4. **MBR_TYPE_CD_ID** -> 일반회원 = **531**
5. **JOIN_METHOD** -> allcone은 "MOBILE"
6. **ENT_ID** -> null로 두고 TODO 주석 명시
7. **DB 위치** -> chainportaldb.tb_member (같은 DB, datasource 변경 불필요)
8. **bctransdbx FK** -> user_regist_truck 등의 USER_ID는 tb_member.LGN_ID와 JOIN

## 미결 사항 (담당자 확인 대기)

1. **BIRTH_DT** - 신 테이블에 컬럼 없음. 로그인 응답에 포함되어 모바일 앱 사용 중. 제거/유지 결정 필요
2. **추천인(RECMMDR_ID, RECMMDR_TRUCK_NO)** - 신 테이블에 컬럼 없음. 가입 시에만 저장, 이후 미사용
3. **DDL에 없는 배치 코드 필드 4건** - birthDate, thirdPartyProvideAgreeYn, individualLocationInfoProcessAgreeYn, lockDate

## 변경 대상 Mapper XML (8개)

1. **UserInfoMapper.xml** - 회원 CRUD 전체
2. **UserSignMapper.xml** - 로그인/로그아웃
3. **DeviceAuthMapper.xml** - SMS 인증
4. **DeviceTokenMapper.xml** - 디바이스 토큰 (JOIN)
5. **AlarmMapper.xml** - 알람 (JOIN)
6. **UserTruckMapper.xml** - 차량 등록 (JOIN)
7. **ChainportalUserMapper.xml** - 체인포털 사용자 (JOIN)
8. **OldDeviceTokenMapper.xml** - 레거시 (JOIN)
